## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/hgbaodev/movieapi.git
   ```

2. Change into the project directory:

   ```bash
   cd movieapi
   ```

3. Run app
   ```bash
   npm install
   ```

4. Run app
   ```bash
   npm run dev
   ```
